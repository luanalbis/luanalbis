## 📌 Projetos em Destaque

### ♟️ [Online Chess WebSocket](https://github.com/luanalbis/online-chess-websocket)
Servidor back-end para partidas de xadrez online em **tempo real**, utilizando **Spring WebSocket** para comunicação bidirecional. Implementa toda a lógica do jogo com POO, validação completa das regras, sistema de **matchmaking** para pareamento automático e suporte a partidas amistosas.
`Java 17` · `Spring Boot` · `Spring WebSocket` · `POO` · `Maven`

---

### 🏭 [Stock Control](https://github.com/luanalbis/stock-control)
Aplicação **fullstack** para gestão de matérias-primas e otimização de produção industrial. O back-end expõe uma API REST completa com algoritmo que sugere quais produtos fabricar para **maximizar o valor total de vendas**, resolvendo conflitos de recursos compartilhados. Inclui testes no back-end e no front-end.
`Java 21` · `Spring Boot` · `H2` · `JUnit 5` · `Mockito` · `Vue 3` · `TypeScript` · `Axios` · `Vitest`

---

### 💸 [Desafio PicPay Simplificado](https://github.com/luanalbis/desafio-backend-picpay-simplificado)
Implementação do famoso desafio técnico do PicPay — uma plataforma de pagamentos com transferências entre usuários e lojistas. Inclui validação de regras de negócio, transações atômicas com `@Transactional`, consulta a **serviço autorizador externo**, envio de notificações e cobertura com **testes unitários** (JUnit 5 + Mockito).
`Java 17` · `Spring Boot 3` · `Spring Data JPA` · `H2` · `JUnit 5` · `Mockito`

---

### ⏱️ [Pomodoro React](https://github.com/luanalbis/pomodoro-react) · [🔗 Demo](https://pomodoro-react-kappa.vercel.app/)
Aplicativo de produtividade baseado na técnica Pomodoro com alternância automática de ciclos, múltiplos temas, histórico de tarefas (completas, pausadas ou abandonadas) e uso de **Web Workers** para o timer não bloquear a thread principal.
`React` · `TypeScript` · `Vite` · `Web Workers` · `Vercel`

---

### 🛒 [Shopping List](https://github.com/luanalbis/shopping-list) · [🔗 Demo](https://shopping-list-ruby-six.vercel.app/)
App web para gerenciamento de listas de compras com criação, edição, visualização de múltiplas listas e checklist interativo de itens com acompanhamento de progresso.
`React` · `TypeScript` · `CSS` · `Vercel`

---

### 🐾 [CRUD de Cadastro de Pets](https://github.com/luanalbis/crud-cadastro-pets-java.io)
Sistema de cadastro de pets via terminal em **Java puro**, com persistência em arquivos `.txt`, validação rigorosa de campos, busca por múltiplos atributos e formulário dinâmico que permite adicionar novas perguntas ao cadastro em tempo de execução.
`Java 17` · `POO` · `Collections` · `Java IO` · `MVC`

---

### 🌍 [REST Countries API + Theme Switcher](https://github.com/luanalbis/rest-countries-api-with-color-theme-switcher-master)
Aplicação que consome a REST Countries API com filtro por região, busca por nome e alternância de tema claro/escuro.
`JavaScript` · `REST API` · `CSS`
